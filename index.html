Aussie Add-ons

Kodi repository of add-ons for Australian content.

We provide Kodi add-ons for all free-to-air TV providers in Australia and some on-demand and live (subscription) for AFL, NRL and Netball.

Most content is geo-blocked, so you will need to be in Australia, or have a suitable VPN provider for access.

Get in touch

We have a Slack workspace set up, which you can use chat with us, and others also using our add-ons. Use the sign-up page and follow the instructions in your email.

We track bugs, issues and feature requests through each add-on's own GitHub issue tracker. You can find a link for the issue tracker in each of the add-ons pages.

Follow @aussieaddons on Twitter for the latest Aussie Add-ons updates.
